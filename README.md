# CAPSTACK
A smart financial safety platform that unifies income and expenses, automates monthly savings, allocates money into emergency and investment buckets, and alerts users during salary delays or layoffs. It helps salaried individuals stay financially stable in an unpredictable job market.

## Project Structure

- `backend-api/`: Node.js Express backend
- `ml-service/`: FastAPI ML service
- `frontend/`: Next.js React frontend
- `database/`: Database migrations and seeds
- `infra/`: Infrastructure configurations
- `analytics/`: Jupyter notebooks and reports
- `docs/`: Documentation files
